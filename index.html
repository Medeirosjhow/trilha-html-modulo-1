O que são Clients?

Em um âmbito geral, cliente é todo aquele que consome algo e para o contexto da internet não é diferente, os clientes são as máquinas e dispositivos que consomem a internet.
Todo dispositivo que está conectado a uma rede, que faz uma requisição e recebe uma resposta.
Para exemplificar, vamos observar o que acontece quando navegamos no google:

Abrimos nosso navegado, digitamos o endereço e pressionamos enter, neste momento nossa máquina “Client” ira fazer uma requisição para um “server”, que ira retornar a página de busca.

O que são Server's

Server’s é o mesmo que servidor, é uma máquina responsável por armazenar conteúdo e servir ele quando for solicitado, podemos comparar com uma lanchonete de esquina.

Em casos mais simples, você pode entrar e pedir uma coxinha pronta, mas se quer complexidade de sabor, você pode pedir um lanche, e na construção do seu lanche haverá uma comunicação do cozinheiro para melhorar o seu lanche, exemplo, quer katcup? Sem mostarda?... etc...
No contexto da internet não é diferente, se o que você deseja é um conteúdo simples como acessar o google, basta pedir no endereço google.com.br que a página será exibida.
Mas se deseja criar uma conta no google, vai precisar fornecer dados e informações para o servidor de forma que ele consiga te servir uma conta.

O que é HTML

HTML ou Linguagem de marcação de texto, é a linguagem padrão da internet, ela é utilizada para marcar o conteúdo de um site de forma que ele seja exibido para o cliente de forma estruturada. A página web é um documento e por isso ela precisa de uma formatação.

Estrutura base de um HTML

<!DOCTYPE html>
<html lang="pt-br" >
<head >

<meta charset="UTF-8" >
<meta http-equiv="X-UA-Compatible" content="IE=edge" >
<meta name="viewport" content="width=device-width, initial-scale=1.0" >
<title>Document </title >
</head >
<body >
</body >
</html >

<!DOCTYPE html>
 Está tag é repsonsável por dizer ao navegador que é um documento do tipo HTML.

<html lang="pt-br">
 Está é uma tag que marca todo o conteudo html, se observar bem, verá que ela começa com html e termina com /html, desta forma marcando todo conteúdo.

<head>
 Está tag é responsável por abraçar toda a cabeça do site, aqui entram alguns metadados como o titulo da página, folhas de estilos e alguns outros arquivos e configurações.

<meta charset="UTF-8">
 Está é uma tag de configuração, aqui dizemos ao navegador qual é o formato de codificação de caracteres utilizado.

<meta http-equiv="X-UA-Compatible" content="IE=edge">
 Tag de compatibilidade com internet explorer

<meta name="viewport" content="width=device-width, initial-scale=1.0">
 Está tag é responsavel por controlar a exibição do nosso site em dispositivos móveis.

<title>Document </title>
 Esta tag é utilizada para marcar o titlo do site, o que aparece na aba do navegador.

</head>
 Tag de fechamento da cabeça do nosso html.

<body>
 Está é tag que irá exibir nosso conteúdo, ela abraça nosos paragrafos imagens e todo tipo de midia.

</body>
 tag de fechamento do corpo do conteudo.

</html>
 tag de fechamento do documento html.

Conhecendo algumas Tags

conheçam alguns elementos de formatação, para exibir tipos especiais de texto.:

  • <a> - Tag utilizada para marcar um link.
  • <abbr> - Está tag é utilizada para marcar uma sigla como HTML. ela adiciona a descrição via propriedade title.
  • <b> - Está Tag é utilizada para marcar um conteúdo como negrito.
  • <blockquote> - Está Tag é utilizada para criar citações no texto.
  • <strong> - Está Tag é praticamente igual ao b sua diferença está no valor semântico, pois ela é responsável por dar uma forte ênfase no texto, fundamental para leitores de tela e acessibilidade.
  • <del> - Está Tag é utilizada na correção de textos: exemplo.
  • <em> - Tag utilizada para dar uma enfase no conteudo, geralmente deixa ele em italico
  • <font> - Tag necessita de atributos, é uma tag de estilização. recomenda-se não usar ela, apenas css.
  • <i> - Está tag é utilizada para deixar o conteudo em italico.
  • <ins> - Define um texto que foi inserido
  • <li> - Define um item da lista
  • <mark> - Marca um texto como se forsse marcado por marcatextoExemplo
  • <ol> Define uma lista ordernada
  • <p> Define um paragrafo
  • <sub> define um conteudo a baixo da linha, ex: h2o
  • <small> Define um texto menor
  • <sup>Define um texto um pouco mais alto, como potenciação: 22
  • <u> Marca um texto com underline
  • <ul> Marca uma lista não ordenada.
